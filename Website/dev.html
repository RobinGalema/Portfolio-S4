Development

Op deze pagina staan de opdrachten etc. beschreven waarmee ik de leeruitkomsten van DEV aantoon.
Hieronder heb ik de leeruitkomsten van DEV nog even kort weergegeven:

  • Architectuur: De student heeft een applicatie gerealiseerd met een module pattern. Externe libraries zijn gebruikt om de applicatie te ontwerpen. De applicatie is tot stand gekomen via user stories en een schets van de architectuur.
  • Beheerbaarheid: De student gebruikt Git gedurende het hele traject. De code en Git-project is duidelijk beschreven volgens gangbare richtlijnen in de praktijk.
  • Ontwikkelomgeving: De student heeft een onderbouwde (front-end) ontwikkelomgeving samengesteld en laat zien op de hoogte te zijn van de in het werkveld gangbare tools.

Hieronder staat al mijn gemaakte werk voor DEV in een lijst weergegeven. Door op een kop te klikken verschijnt er een beschrijving met reflectie van het gemaakte werk met daarbij de bijbehordende leeruitkomst.
